ejemplos de aplet con java

Un applet conforme a lo que encontramos en las páginas de Oracle: "es una aplicación hecha en Java que se llama desde un documento HTML y se ejecuta dentro de un navegador”. Es necesario recordar que el navegador debe tener instalado el plugin del la Máquina Virtual de Java (JVM). Máquina Virtual de Java: Es el programa que se encarga de ejecutar el bytecode generado por el compilador de Java. Para entenderla foma e que s manejan os applets vamos a ver un ejemplo que presenta el clásico Hola mundo



import java.awt.Graphics;
import javax.swing.JApplet;
public class HolaMundo extends JApplet {
public void paint (Graphics g) {
super.paint (g);
g.drawString("Hola mundo!!",10,20);
}
}
