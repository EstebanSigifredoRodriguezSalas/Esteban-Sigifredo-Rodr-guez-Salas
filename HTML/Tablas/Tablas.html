Marcos de trabajo Java

La palabra inglesa "FRAMEWORK"(marco de trabajo) define, en terminos generales, un conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como de índole similar.

En el desarrollo de software un "FRAMEWORK" o infrestuctura dÍgital, es una estructura conceptual y tecnologica de soporte definido, normalmente con artefactos o mudulos de software concretos, que puede servir de base para la organización y desarrollo de software. Típicamente hablando, puede incluir soporte de programas, bibliotecas, y un lenguaje interpretado, entre otras herramientas, para así ayudar a desarrollar y unir componentes de un proyecto.
